

* 指令格式

={7.h}{4.a}xxx{8.r}{1.v}{5.s}{6.i}_{2.s,u}_{3.b,h,w,d}=

- 1.v
- 2.s,u  signed/unsigned
- 3.b,h,w,d 向量元素的类型: int8, int16, int32, int64
- 4.a absolute
- 5.s saturate
- 6.i 立即数
- 7.h horizontal
- 8.r rounding
